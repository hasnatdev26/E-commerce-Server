#  MERN E-Commerce Website

A modern **E-Commerce web application** built with the **MERN Stack (MongoDB, Express, React, Node.js)**.
Users can browse products, view details, and place orders easily.

##  Live Demo

🔗 https://classy-mousse-8b9f13.netlify.app

##  Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* Tailwind CSS
* TanStack Query
